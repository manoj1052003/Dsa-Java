# Java DSA — Repository

A collection of Java projects and practice problems organized by topic.  
This repository contains full Eclipse project folders — import individual projects into Eclipse using **File → Import → Existing Projects into Workspace**.

---

## Repository structure

- **Arrays/** — Array problems, basic DSA practice, placement questions, LeetCode-style exercises.
- **Strings/** — String manipulation problems and utilities.
- **Sorting/** — Implementations and examples of sorting algorithms (merge, quick, bubble, selection, insertion).
- **Trees/** — Tree data structures and tree-based problems (binary tree, BST, traversals).
- **Graphs/** — Graph algorithms and practice projects.
- **Recursion/** — Recursive and backtracking practice problems.
- **JDBC/** — Database/DAO examples and JDBC demo projects (MySQL connector libs may be inside project `lib/` folders).
- **JavaBasics/** — Small Java learning projects, tutorials and sample programs.
- **Others/** — Miscellaneous projects, utilities, and experiments.
- **.metadata/** — Eclipse workspace metadata (kept locally; not recommended to share). See `.gitignore`.
